Предсказание значения курса бел. рубля (BYN) по отношению к амер. доллару (USD).
В работе используется стоимость 1 USD в бел. рублях.

В ходе выполнения работы было сформировано 2 модели, построенные на основе рекуррентных искусственных нейронных сетей (ИНС):

	PyCharm-проект первой модели находится в директории "NextValuePrediction".
	Модель решает задачу без доп. обработки входных данных (в обоих случаях это показатели стоимости 1 амер. доллара в бел. рублях с 2012 г. по 2021 включительно, за 10 лет).
	Т. е. предсказание значения курса производится на основе n предыдущих показателей.
	В качестве иллюстрации приведен график (файл PredictionExamplePlot.png), на котором представлены истинные показатели (за 2022 г.) одновременно с предсказанными полученной ИНС (последние 50 дней).

	PyCharm-проект второй модели находится в директории "OutputValuePrediction".
	Данная модель ИНС требует доп. обработки входной выборки данных - данные из выборки формируют "свечное представление".
	Т. е. для каждого периода в n дней (в работе использовалась неделя, n = 7) на вход ИНС подается след. информация: начальная цена (на 1-й день), макс. и мин. цена (за n дней) соответственно.
	Модель предсказывает выходную цену (на n-й день).
	В качестве иллюстрации реальных выходных цен в конце каждой недели (2022 г.) и предсказанных аналогов также приведен график (файл PredictionExamplePlot1.png).
